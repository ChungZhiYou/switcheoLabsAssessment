# Three ways to sum n

<h2>sum_to_n_a</h2>

-   For loop
-   time complexeity: O(n)
-   Space Complexeity: O(1)
<br>

<h2>sum_to_n_b</h2>

- Recursive Function
- time complexeity: O(n)
- Space Complexeity: O(n)
<br>

<h2>sum_to_n_c</h2>

- While loop
- time complexeity: O(n)
- Space Complexeity: O(1)
